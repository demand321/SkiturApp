# SkiturApp

A mobile app for organizing ski touring trips with friends. Plan trips, track routes in real-time, share photos, chat, and keep a full trip archive.

## Features

- Trip planning with invitations (SMS/link for non-users)
- Real-time GPS route tracking (works offline)
- Geotagged photos with captions
- Real-time group chat per trip
- Weather forecasts from Yr.no (MET Norway)
- Push notifications for all trip events
- Shopping list for trip supplies
- Full trip history and statistics

## Tech Stack

- **Mobile:** React Native (Expo) + TypeScript
- **Backend:** Firebase (Auth, Firestore, Storage, Cloud Functions, FCM)
- **Maps:** Mapbox GL
- **Weather:** MET Norway Locationforecast 2.0
